# minetoot

## Summery
Minecraftのログを読み込んで、何かログが追加されたら日本語に変換したうえで Mastodon にTootできるプログラムです。

## Usage
[Twitter の投稿を IFTTT で Mastodon へ転送する。 - Qiita](http://qiita.com/yukimochi/items/f80a50a4486d0cb770dc)
の、ユーザーに対するアプリケーション認証まで行ってください。
